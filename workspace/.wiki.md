# Project Summary
The Ministry of Coffee project aims to create a multilingual website for a cafe located in Warsaw, Poland. This static site will provide information about the cafe's offerings, accessibility features, and location, catering to Polish, English, Turkish, and Ukrainian-speaking customers. The website is designed to be deployed on GitHub Pages without requiring additional configuration.

# Project Module Description
The project consists of several functional modules:
- **Hero Section**: Displays the cafe name, logo, and a call-to-action button.
- **About Section**: Provides details about the cafe, including its description, rating, and opening hours.
- **Features Section**: Highlights services, amenities, and popular offerings.
- **Gallery Section**: Showcases images of the cafe, including its interior and exterior.
- **Location Section**: Displays the cafe's address, phone number, and an embedded Google Map.
- **Footer**: Contains social media links and copyright information.

# File Description Inventory
- **public/assets/**: Contains images used in the website.
- **src/components/**: Contains React components for different sections of the website.
- **src/lib/translations.ts**: Holds multilingual content for the website.
- **src/pages/**: Contains the main and not found pages.
- **App.tsx**: Main application component that sets up routing and context.
- **main.tsx**: Entry point for the React application.
- **index.html**: HTML template for the website.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Shadcn-ui**: Component library for UI elements.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for fast development and production builds.

# Usage
To install dependencies, build, and run the project:
1. Clone the repository.
2. Navigate to the project directory.
3. Run `pnpm install` to install dependencies.
4. Run `pnpm run build` to create a production build.
5. Deploy the contents of the `/dist` folder to GitHub Pages.
